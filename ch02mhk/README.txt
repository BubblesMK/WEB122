Objective:
Apply Chapter 2 CSS concepts by using proper sizing units, nested lists, custom properties, media queries, and viewport‑based masthead design.

Requirements:
HTML

•  Include a meta tag with your name as the author:
<meta name="author" content="Your Name">
•  Use semantic tags: header, nav, main, footer
•  Use appropriate h1–h6 tags to outline text content
•  Use article, section, and aside where needed
•  Use div tags sparingly

CSS Requirements (all require commenting)

•  Use px, em, and rem sizing values based on best practices
•  Create a nested unordered list 4 levels deep using em without causing shrinking
•  Create a second nested unordered list 4 levels deep using rem without causing shrinking
•  Create a media query for tablet and desktop that changes font-size and background-color
•  Create 3 custom properties (variables)
•  Create a masthead using viewport units

Optional

•  Use line-height
•  Use calc() or clamp()
